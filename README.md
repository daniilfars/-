Суть игры: генерируется поле 5 х 5 с пятью минами. Игрок вводит координаты клетки, которую хочет открыть. Если клетка не была заминирована(0), то выводится количество мин вокруг этой клетки. А если же была заминирована(-1), То он проиграл, и выводится начальное поле с минами. Игрок выиграет, если откроет все не заминированные клетки.
